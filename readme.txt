== Description ==

Display dynamic phone numbers based on URL query strings, form POST fields, or cookie data.

You can find the source code at the GitHub repository:
https://github.com/ninthlink/nlk-plugins/tree/master/dynamic-phone


== Installation ==

1. Upload file to the `/wp-content/plugins/` directory
2. Activate the Dynamic Phone Numbers plugin through the 'Plugins' menu in the WordPress Admin Panel.
3. Go to the new "Dynamic Phone" menu item in the left admin menu to configure to your desired specifications.
